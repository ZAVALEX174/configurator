Возможности конфигуратора

Собственные проекты и схемы

Неограниченное количество собственных проектов, загрузка собственного плана проекта для расстановки монтажных блоков.

Удобное наполнение онлайн

С помощью удобного онлайн интерфейса выбираем нужный бренд, серию, цвет и форму элементов и наполняем монтажные блоки, перетаскивая элементы на блок мышкой (Drag And Drop)

Управление монтажными блоками

обавление, удаление, копирование монтажных блоков. Массовые операции с блоками: изменение свойств сразу нескольких блоков, удаление группы блоков.

Сборка монтажного блока под ключ

Доступны для наполнения элементы всех необходимых функциональных групп: клавиши, розетки, информационные разъемы, диммеры, декоративные накладки и т.п.

Помещения на схеме

Рисуем отдельные помещения выбранным цветом на своём плане проекта, чтобы задать общие свойства монтажным блокам на уровне помещений и быстро добавлять новые блоки с заданными у помещения свойствами.

Монтажные стандарты

Сборка монтажных блоков с учетом разных монтажных стандартов: итальянский стандарт, немецкий стандарт.

Личный кабинет

Неограниченное количество собственных проектов, только вы имеете доступ к своим проектам.

Автоматическая генерация спецификаций

В личном кабинете доступно автоматическое формирование спецификаций проекта по вашему наполнению: графическая спецификаций, раскладка по помещениям, номенклатурная спецификация.
